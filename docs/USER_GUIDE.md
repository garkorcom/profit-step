## 📋 GTD — Управление задачами

### Колонки доски

| Колонка | Назначение |
|---------|------------|
| **Inbox** | Входящие задачи |
| **Next Actions** | К выполнению сейчас |
| **Projects** | Сложные задачи |
| **Waiting For** | Ожидание от других |
| **Estimate** | Требует оценки |
| **Someday** | Когда-нибудь |
| **Done** | Завершённые |

### Добавить задачу
1. Нажмите `+ Add Task`
2. Выберите клиента
3. Выберите категорию (Купить, Проверить и т.д.)
4. Введите описание
5. Нажмите "Сохранить"

### Запустить таймер из задачи
1. Наведите на задачу
2. Нажмите ▶ (зелёная кнопка)
3. Таймер запущен!

## ⏱ Учёт времени

### Просмотр сессий
1. Перейдите в `/crm/timetracking`
2. Используйте фильтры по дате/сотруднику
3. Кликните на сессию для деталей

### Редактирование (для админов)
1. Кликните на сессию
2. Измените время/ставку
3. Сохраните

---

## 🛒 Закупки

### Создать список
1. Перейдите в `/crm/shopping`
2. Нажмите `+ Добавить список`
3. Выберите клиента
4. Добавьте товары

### Загрузить чек (через бот)
1. В боте: `🛒 Закупки`
2. Выберите товары
3. Отправьте фото чека
4. Подтвердите сумму
5. Отправьте фото товаров

## ❓ Частые вопросы

**Q: Как изменить пароль бота?**
A: Обратитесь к администратору

**Q: Не работает геолокация?**
A: Разрешите Telegram доступ к геолокации

**Q: Задачи не создаются из голосового?**
A: Говорите чётко, используйте фразы "нужно сделать...", "завтра..."

**Q: Где посмотреть мой заработок?**
A: В боте после завершения смены, или в веб-панели `/crm/timetracking`
