# Landing Pages (Idea Hub)

Здесь хранятся все посадочные страницы компании, которые служат "инкубаторами идей" для проверки гипотез, продажи услуг и генерации лидов.

## Архитектура папки

В целях изоляции и ускорения разработки, все лендинги изолированы от логики основного CRM-приложения.

- `/infrastructure/` — глобальные переиспользуемые элементы для лендингов.
  - `shared.css` — единые стили (цвета, переменные).
  - `/components/` — общие компоненты для конверсии (кнопки, формы захвата).
  - `/icon-kit/` — иконки в SVG-формате или шрифтах.

- `/ideas/[имя_проекта]/` — папка конкретного лендинга:
  - `/concept/` — ТЗ, референсы, заметки (сырые данные).
  - `/build/` — код самого лендинга (точкой входа обязательно должен быть `index.tsx`!).

## Инструкция по созданию нового лендинга

1. Создайте папку в `/ideas/ваше-название`.
2. Создайте папку `/build` внутри и файл `index.tsx`. 
3. В файле нужно обязательно сделать экспорт по умолчанию (`export default function Landing() { ... }`).
4. Для отображения меты в админке добавьте именованный экспорт метаданных:
   ```tsx
   export const metadata = {
     title: 'Название',
     description: 'Описание лендинга',
     status: 'Development', // Опционально
   };    
   ```
5. Ваш лендинг будет автоматически доступен публично по URL `/l/ваше-название` и появится в админ-панели на вкладке "Лендинги".
