## ⏱ Учёт рабочего времени

### Начать работу
```
1. Нажмите ▶️ Start Work
2. Выберите клиента
3. Подтвердите геолокацию (опционально)
4. Запишите голосовое с планами на день
```

### Завершить работу
```
1. Нажмите ⏹️ Finish Work
2. Подтвердите геолокацию
3. Запишите голосовое с отчётом
4. (Опционально) Загрузите фото работ
```

### Перерыв
```
☕ Break — приостановить таймер
▶️ Resume Work — продолжить
```

## 🎙 AI-создание задач

Бот автоматически распознаёт задачи из голосовых отчётов!

### Пример
Вы говорите:
> "Сегодня закончил плитку. Завтра нужно купить затирку и доделать двери."

AI создаёт задачи:
- ✅ Купить затирку
- ✅ Доделать двери

### Как работает
1. Запишите голосовое при завершении работы
2. AI (Gemini) транскрибирует аудио
3. Извлекает задачи, даты, приоритеты
4. Создаёт задачи в GTD с меткой `@bot`

## 🛒 Модуль закупок

### Меню покупок
Нажмите `🛒 Закупки` в главном меню

### Функции
- **Выбрать товары** — отметить товары из списка
- **Загрузить чек** — фото чека из магазина
- **Фото товаров** — подтверждение покупки (Double Proof)
- **Добавить товар** — новый товар голосом/текстом

### Double Proof (Двойное подтверждение)
```
1. Выбрать товары из списка
2. Сфотографировать чек
3. AI распознаёт сумму
4. Подтвердить сумму
5. Сфотографировать купленные товары
6. ✅ Закупка завершена
```

## ❓ FAQ

**Q: Как изменить ставку?**
A: Ставка устанавливается администратором в веб-панели

**Q: Задачи не создаются из голосового?**
A: Убедитесь что вы чётко проговариваете действия. AI распознаёт фразы типа "нужно...", "завтра...", "сделать..."

**Q: Не отправляется геолокация?**
A: Проверьте разрешения Telegram на доступ к геолокации
